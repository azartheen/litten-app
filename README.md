# litten-ios

litten app iOS repository using React Native

## Development Commands

```sh
# Install dependencies
yarn install

# Install the cocoapods dependencies
cd ios/
pod install
cd -

# Run the simulator
yarn ios

# Run the ES linter
yarn lint

# Run the tests
yarn test

# Prepare the data dependencies
yarn prepare-data
```
